# Mac Dev Env Setup

You can run `mac-installer.sh` and after you getting a new MacOS by:

```sh
wget -O mac-installer.sh "https://github.com/evecalm/mac-env/raw/master/mac-installer.sh" && sudo bash mac-installer.sh
```
